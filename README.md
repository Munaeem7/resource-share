Resource Sharing Platform
A full-stack web application that enables users to securely share and access study resources. The platform supports authentication, profile management, and real-time resource handling.

Features

Secure user authentication using Firebase (Email/Password and Google OAuth)
User profiles with dashboard
Upload and download study materials
Download count tracking
Responsive design with real-time updates

Tech Stack
Frontend: React, Tailwind CSS
Backend: Node.js, Express
Database: MongoDB Atlas
Authentication: Firebase

Purpose
This project demonstrates core full-stack development concepts including authentication, REST APIs, database management, and responsive UI design.
